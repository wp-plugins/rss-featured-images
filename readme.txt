Inserts featured images from your posts into your site's RSS feed. Allows for featured image size, positioning and, custom format options.

== Description ==

<p>Allows you to not only add featured images from your posts to your site's RSS feed but to select the size of the image, specify the images alignment, and to optionally create your own custom format for your featured images.</p>

<p>Please use the support tab to submit your questions, concerns, or enhancement requests.</p>



== Installation ==

<h4>Automatic installation</h4>

<p>Automatic installation is the easiest option - WordPress handles the file transfers so you don’t need to leave your web browser. To perform an automatic install login to your WordPress dashboard, navigate to the Plugins menu and, click Add New.</p>

<p>In the search field type “RSS Featured Images” and click Search Plugins. Once you’ve found the plugin you can install it by clicking “Install Now”.</p>

<h4>Manual installation</h4>

<p>The manual installation method involves downloading the plugin and uploading it to your webserver via an FTP application. See the WordPress codex for 
<a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to do this</a>.</p>
